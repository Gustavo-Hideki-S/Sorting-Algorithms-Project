# Sorting Algorithms Project

This project was developed as part of an academic assignment. It features graphical implementations of several sorting algorithms, including:

- **Merge Sort**  
- **Selection Sort**  
- **Shell Sort**  
- **Insertion Sort**

The code is functional, but there are plans for improvements in future commits.

## Next Steps / Planned Updates
